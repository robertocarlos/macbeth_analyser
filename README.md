# MacbethAnalyser

This analyses Macbeth play, counting speechs by persona.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'macbeth_analyser'
```

And then execute:

  $ bundle

Or install it yourself as:

  $ gem install macbeth_analyser

## Usage

  $ ruby lib/macbeth_analyser.rb
  
## Running tests

  $ rspec
